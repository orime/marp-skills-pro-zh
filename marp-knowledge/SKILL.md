# Marp Skills Pro - 核心知识库 (Knowledge Base)

本项技能是提供 Marp 框架最全面参考指南的知识大脑，作为生成现代化幻灯片的根基知识。

## 概要

当 AI 需要确认某个高级排版指令（Directive）、或者理解高级毛玻璃拟物化效果（Glassmorphism）和现代 CSS 变量布局时，它将从这里抽调理论基础。无论是排版修复、编写内容还是开发高级主题背景，都需要依赖这里的底层逻辑。

## 查阅文档目录

- [directives.md](references/directives.md) - 全局与局部指令对照表（包含暗黑模式反转 `<!--_class: invert-->`）
- [theme-system.md](references/theme-system.md) - 主题 CSS 架构指南（推荐支持 Flex / Grid 系统）
- [image-syntax.md](references/image-syntax.md) - 强大的排版级画像与背景底图设定语法
- [math-code.md](references/math-code.md) - 数学公式与带有现代阴影轮廓的代码块编写规范

## 触发场景

- 用户询问 Marp 有什么特殊语法或指令时。
- 被其他专业技能 (如 `marp-visual-fix`, `marp-theme-creator`) 作为依赖调用时。
- 需要确认和运用局部指令或主题系统属性构造高质感排版卡片时。
