# Marp Skills Pro - 高端定制主题工厂 (Theme Creator)

这是一套帮助 AI 快速建立符合现代审美标准的高级 Marp 定制主题（SCSS / CSS）的技能。

## 触发场景

- “我想做一个专属公司的主题”
- “帮我写一套苹果发布会那种风格的极简 CSS 主题”
- “我希望能有支持毛玻璃组件的全新色彩排版”
- “统一接下来所有的深色汇报展现格式”

## 高级执行架构流

1. **提取品牌/设计要求**：明确主色调，是否需要启用（全反转暗黑模型 `invert` 或白昼明亮模式）。
2. **高端设计元素植入**：默认使用渐变文字、多维投影底色、毛玻璃 (Glassmorphism) 及无衬线原生字体。
3. **编写 SCSS / CSS 核心库**：快速建立变量与原子类定义。
4. **组件标签配置**：定义类似 `.grid`（两栏均分布局）和 `.glass`（悬浮容器）等内置结构标签供编写 Markdown 时调用。
5. **主题文件打包和示例生成**：提供并构建包含主题指令的首个展示原型。

## 联动的技能节点

- **marp-knowledge**: 控制 Marp 主题语法标准。
- **marp-theme-builder**: 为大型构建配置 CI 流水线测试环境。

## 核心资产

## 设计金库指标 (Pro)
1. **对比分明**：暗色模式采用带有发光环境色的 `#0f172a` 星际蓝/黑色替代纯黑。
2. **多层微距阴影**：卡片 `box-shadow` 不要生硬单层，改用带有柔和扩散系数的多层投影参数。
3. **现代容器**：内置带有 `backdrop-filter: blur(x)` 的 `.glass` 扩展组件。
4. **极致易读**：代码片段采用 `Fira Code` 级别程序员宽等宽字体，加大行距。
