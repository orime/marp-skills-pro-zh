# Marp Skills Pro - 文本与内容创作核心 (Marp Markdown)

用于高效撰写和修改支持 Marp 引擎的 Markdown 文稿的 AI 技能。

## 触发场景

- “帮我写一个推介方案 PPT”
- “帮我创建一个高级幻灯片代码”
- “把这份产品大纲转化为一个高端的演示文稿”
- “给我写一套带科技风的光影幻灯片”

## 执行工作流

1. **需求理解**：精准获取用户演示目标、核心信息和预期观众（以及用户是否有倾向的现代主题方案，如 premium 风格）。
2. **主题方案**：优先推荐或注入带 `glass` 和 `grid` 双列的高阶结构，默认开启深色反转背景适配。
3. **分镜构建**：构思整套多页面幻灯片脚本和分页骨架。
4. **Markdown生成**：通过巧妙使用 `<!--_class: lead/invert-->` 等预设 CSS 组件或 `<div class=\"grid\">` 生成精美的内容模块。
5. **代码查验**：使用预览环境或建议用户部署构建查看效果。

## 引用的附属功能

- **marp-knowledge**: 包含现代主题组件、图片排版语法详细说明。

## 制作黄金守则（Pro）

1. **层级对比化**：主标题、短语要有强烈的大小和粗细错落感。
2. **善用内嵌网格和卡片**：如果内容有左右比较属性，一律采用并排栅格结构，甚至在重点处引入 `<span class="tag">` 小组件突显。
3. **保持图表呼吸感**：每页幻灯片文本项不要超过三要点，把其余空间让给留白或玻璃质感背景卡片处理。
4. **核心页过渡设计**：强制运用 `<!--_class: invert-->` 或 `lead` 进行转折阶段或重要章节的心里暗示。
